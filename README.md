## 使用方式

1. 打开 [templates](./templates) 目录。
2. 按风格、行业、页数选择合适的 `.docx` 文件。
3. 下载后用 Word 或 WPS 修改个人信息、教育经历、实习经历和项目经历。
4. 投递前导出 PDF，避免不同设备打开时排版变化。
5. 转发或二次整理时请保留 HICV.cn 来源信息。
